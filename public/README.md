# Public Assets

This directory contains static assets like images and the resume PDF.

## ⚠️ IMPORTANT: Resume File
The app's "Download CV" links expect a file named **`SUNAM_KUNDAL_RESUME.pdf`** in this directory.

Drop your latest resume here and keep the filename exactly `SUNAM_KUNDAL_RESUME.pdf`
(referenced from `src/components/Hero.tsx` and `src/components/Navigation.tsx`),
otherwise the download link will 404.

## Profile photo
`profile.png` / `profile.webp` are the hero profile images. Replace them with your own
photo (keep the same filenames) — they are preloaded as the LCP image in `index.html`.
